# RTSP Face Detection — FFmpeg GPU + TensorRT
**Author:** Anukool Kashyap  
RTSP live camera face detection using FFmpeg GPU decode (NVDEC) + TensorRT YOLOv8-face.

---

## File Structure

```
RTSP-FFmpeg-TRT/
│
├── main.py                     ← entry point — run this
├── config.py                   ← all settings (RTSP URL, paths, thresholds)
├── requirements.txt
├── README.md
│
├── models/
│   ├── yolov8n-face.pt         ← download separately
│   ├── yolov8n-face.onnx       ← generated by convert script
│   └── yolov8n-face.engine     ← generated by build_engine.py ← used at runtime
│
├── utils/
│   ├── ffmpeg_capture.py       ← RTSP + GPU decode via PyAV (libav)
│   ├── trt_inference.py        ← TensorRT engine loader + YOLOv8 inference
│   └── frame_saver.py          ← saves detected frames + JSON metadata
│
├── scripts/
│   ├── check_gpu.py            ← verify GPU setup before running
│   └── test_camera.py          ← test RTSP connection + GPU decode
│
├── captured_faces/             ← saved frames with detections (auto-created)
└── logs/                       ← run logs (auto-created)
```

## How the GPU pipeline works

```
Camera (RTSP)
    ↓
libavformat (PyAV)      — opens RTSP, reads compressed H264 packets
    ↓
libavcodec h264_cuvid   — sends H264 to NVDEC chip on GPU
    ↓
NVDEC (GPU chip)        — decodes H264 → NV12 frame in VRAM
    ↓
libswscale              — NV12 → BGR, frame comes to RAM as numpy array
    ↓
TRT preprocess          — resize 640×640, normalize
    ↓
TensorRT engine         — yolov8n-face inference
    ↓
bounding boxes          — saved to captured_faces/
```

## Setup

### Step 1 — Install packages

```bash
pip install torch --index-url https://download.pytorch.org/whl/cu121
pip install av
pip install tensorrt-cu12 --extra-index-url https://pypi.nvidia.com --trusted-host pypi.nvidia.com
pip install pycuda
pip install ultralytics opencv-python numpy Pillow tqdm
```

### Step 2 — Check GPU setup

```bash
python scripts/check_gpu.py
```

All items should show ✓.

### Step 3 — Build TensorRT engine (run once)

```bash
python build_engine.py
```

Generates `models/yolov8n-face.engine`. Takes 5–15 minutes.

### Step 4 — Test camera connection

```bash
python scripts/test_camera.py --url "rtsp://admin:admin@192.168.1.64/stream1"
```

### Step 5 — Edit config.py

Set your `rtsp_url` and confirm `engine_path`.

### Step 6 — Run

```bash
python main.py
```

## Key concepts

| Term | What it means |
|------|--------------|
| PyAV | Python wrapper around FFmpeg's libav libraries |
| libavformat | Opens RTSP stream, reads H264 packets |
| libavcodec | Decodes H264 — uses NVDEC when GPU mode enabled |
| h264_cuvid | The codec name that routes decode to NVDEC |
| NVDEC | Dedicated H264 decode chip on NVIDIA GPU |
| NV12 | Pixel format NVDEC outputs — a YUV variant |
| AVHWDeviceContext | CUDA bridge between libav and GPU |
| TensorRT | Compiles YOLO model to GPU-optimised engine |
